# RSGymPT

**RSGymPT** is an application designed to manage at-home personal trainer sessions for RSGym. The platform provides features such as user authentication, booking management, and a personal trainer directory.

## Features
- User login and authentication
- Personal trainer session booking and management
- Directory of personal trainers
- Administrative tools for users and sessions

## About
This project is developed as part of a learning exercise to practice object-oriented programming principles and apply software design best practices.
